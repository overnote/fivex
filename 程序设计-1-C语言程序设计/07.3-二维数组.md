# 07.3-二维数组

## 一 二维数组概念

二维数组常称为矩阵（matrix），定义：`类型说明符 数组名[常量表达式][常量表达式]`

```c++
float arr1[3][6];           // 第1维有3个元素，第2维有6个元素
```

注意：二维数组的元素在内存中人人干事连续线性存放的，不是抽象出来的矩阵形象。

二维数组初始化：

```c++
int a[3][2] = {{1,2},{5,6},{9,7}};
int b[3][2] = {1,2,5,6,7,9}; // 效果和上一行一致，但是表述不清晰
int c[3][4] = {{1},{5},{9}}; // 也可以部分赋值，其余自动为0
```
